Chapter 1: What JavaScript Means

The Most Important Language for the Web

console.log("Hello, World!");

Chapter 2: Variables and Data Types

You can use var, let, and const to store data.

let name = "Ali";
const age = 25;
console.log("Name: " + name + ", Age: " + age);

Chapter 3: Operators and Expressions

Operators are used to perform calculations or evaluate data.

let x = 10;
let y = 5;
console.log(x + y); // 15
console.log(x > y); // true

Chapter 4: Functions in JavaScript

Functions allow you to reuse your code.

function greet(name) {
return "Hello, " + name + "!";
}
console.log(greet("Amina"));

Chapter 5: Conditional Statements

Conditional statements are used for decision-making in code.

let age = 20;
if (age >= 18) {
console.log("You are an adult.");
} else {
console.log("You are a minor.");
}

Chapter 6: Loops in JavaScript

Loops help you execute repetitive tasks easily.

for (let i = 1; i <= 5; i++) {
console.log("Number: " + i);
}

Chapter 7: Arrays and Objects

Arrays and Objects are structured ways to store data.

let fruits = ["Apple", "Banana", "Mango"];
let person = { name: "Khadar", age: 30 };
console.log(fruits[1]); // Banana
console.log(person.name); // Khadar

Chapter 8: DOM Manipulation

DOM manipulation allows you to modify HTML and CSS using JavaScript.

document.querySelector("h1").textContent = "Welcome to JavaScript!";
